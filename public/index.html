This is the API page for the WorkWeek app

Requests:

'/signup: POST request - send email, password, workplace name and admin bool to create a new user in the DB

'/signin': POST request - send email and password to recieve user data and login key

'/users': GET/POST request - authentication required, get or edit data for a specific user

'/workplaces': GET/POST request - authentication required, get or edit data for a specific workplace
